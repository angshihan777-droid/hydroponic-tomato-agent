# 环境调控资料

## 空气湿度偏高

chunk_id: environment_high_humidity
category: environment

适用场景：温室内相对湿度长期偏高，叶面容易结露，通风不足，或夜间湿度明显升高。
风险说明：高湿环境会延长叶面湿润时间，增加灰霉病和其他叶部病害发生风险。
管理建议：加强通风换气，合理控制夜间温差，避免叶面长时间结露。必要时调整植株间距、整枝打叶和空气循环。
来源：- https://extension.umd.edu/resource/botrytis-high-tunnel-tomatoes/

## 温度偏高

chunk_id: environment_high_temperature
category: environment

适用场景：温室白天温度持续偏高，番茄出现萎蔫、坐果下降、花粉活力降低或果实发育异常。
风险说明：高温会增加蒸腾和水分需求，影响开花坐果和果实膨大，还可能间接提高脐腐病风险。
管理建议：检查遮阴、通风、湿帘风机或其他降温设备运行情况。高温时段避免过度水分胁迫，并结合水肥管理保持根区供水稳定。
来源：- https://extension.okstate.edu/fact-sheets/commercial-production-of-fresh-market-tomatoes

## 夜间温度偏低

chunk_id: environment_low_night_temperature
category: environment

适用场景：温室夜温偏低，昼夜温差过大，植株生长变慢，花粉活力下降或果实发育延迟。
风险说明：夜温过低会降低作物代谢速度，影响根系吸收、花器发育和果实膨大。低温叠加高湿时，还会增加叶面结露和病害风险。
管理建议：检查夜间保温、加温设备和通风策略。低温时段避免过度通风，并结合湿度管理减少结露。
来源：- https://extension.okstate.edu/fact-sheets/commercial-production-of-fresh-market-tomatoes

## 光照不足

chunk_id: environment_low_light
category: environment

适用场景：连续阴雨、遮阴过强或温室覆盖材料透光率下降，导致日照时长和光照强度不足。
风险说明：光照不足会降低光合作用，造成植株同化产物不足，表现为生长弱、坐果差、果实膨大慢和品质下降。
管理建议：检查棚膜清洁度、遮阳网使用时长和植株密度。必要时减少不必要遮阴，改善整枝打叶和冠层透光条件。
来源：- https://extension.okstate.edu/fact-sheets/commercial-production-of-fresh-market-tomatoes

## 通风不足

chunk_id: environment_poor_ventilation
category: environment

适用场景：棚内闷湿、空气流动差、二氧化碳交换不足，叶面长时间不干。
风险说明：通风不足会让热量和湿气滞留在温室内，导致高湿、高温或病害风险叠加。
管理建议：检查天窗、侧窗、循环风机和作业通道。优先改善空气交换和冠层内部空气流动，再结合温湿度数据调整通风时段。
来源：- https://extension.umd.edu/resource/botrytis-high-tunnel-tomatoes/
